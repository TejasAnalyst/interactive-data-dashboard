# Interactive Data Dashboard

This project is part of a virtual internship program.

## Description
An interactive data dashboard built using Python and Streamlit to visualize and explore a dataset.

## Tools & Technologies
- Python
- Streamlit
- Pandas

## Features
- Displays dataset in tabular format
- Interactive bar chart visualization
- Simple insights from data

## How to Run
1. Install dependencies:
   pip install streamlit pandas

2. Run the app:
   streamlit run app.py
